# MCP Servers Configuration

This file documents the MCP (Model Context Protocol) servers configured for Claude Code workflows.

## Configured Servers

| Server | Transport | Purpose | Auth Required |
|--------|-----------|---------|---------------|
| `github` | stdio | GitHub API operations (issues, PRs, repos) | PAT |
| `context7` | stdio | Documentation fetching from context7.com | None |
| `sequential-thinking` | stdio | Step-by-step reasoning and problem solving | None |
| `memory` | stdio | Knowledge graph for session memory | None |
| `figma` | HTTP | Figma design context extraction | PAT |

## Server Details

### github

GitHub MCP server for repository operations.

**Setup:**
1. Generate PAT at: https://github.com/settings/tokens
2. Required scopes: `repo`, `read:org` (for private repos)
3. Set `GITHUB_PERSONAL_ACCESS_TOKEN` in `.env` or replace placeholder

**Tools:** `search_repositories`, `create_issue`, `create_pull_request`, `get_file_contents`, etc.

### context7

Documentation fetcher from context7.com/llms.txt sources.

**Setup:** No configuration required.

**Tools:** `resolve-library-id`, `query-docs`

### sequential-thinking

Structured thinking tool for complex problem analysis.

**Setup:** No configuration required.

**Tools:** `sequentialthinking`

### memory

Knowledge graph for persistent entity/relation storage.

**Setup:** No configuration required. Data stored in `~/.claude/memory/`.

**Tools:** `create_entities`, `create_relations`, `search_nodes`, `read_graph`

### figma

Figma MCP server for design context extraction. See [detailed setup guide](../docs/claude/configuration/figma-setup.md).

**Setup:**
1. Generate PAT at: Figma → Settings → Security → Personal access tokens
2. Add to `.claude/.env`:
   ```
   FIGMA_PERSONAL_ACCESS_TOKEN=your_token_here
   ```

**Tools:** `get_file`, `get_file_nodes`, `get_images`, `get_image_fills`

**Token Budget:** Extract specific nodes (<5K tokens), avoid full files (50K+ tokens).

## Environment Variables

| Variable | Server | Required |
|----------|--------|----------|
| `GITHUB_PERSONAL_ACCESS_TOKEN` | github | Yes |
| `FIGMA_PERSONAL_ACCESS_TOKEN` | figma | Yes |

Store in `.claude/.env` (gitignored) or system environment.

## Adding New Servers

1. Add config to `.mcp.json`
2. Update `.mcp.json.example` with placeholder
3. Document in this file
4. Update `docs/claude/configuration/README.md`

## Troubleshooting

### Server not connecting
- Check token is valid and not expired
- Verify environment variable is set
- Restart Claude Code session after config changes

### Rate limiting
- GitHub: 5,000 requests/hour with PAT
- Figma: Standard API limits apply

### Permission errors
- Verify token scopes match required permissions
- Check resource (repo/file) access permissions
