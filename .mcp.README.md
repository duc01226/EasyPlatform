# MCP Servers Configuration

This document describes the Model Context Protocol (MCP) servers configured for Claude Code in this project.

## Configuration File

**Location:** `.mcp.json`

## Configured Servers

### playwright

Browser automation and testing using Playwright.

```json
"playwright": {
  "command": "cmd",
  "args": ["/c", "npx", "@playwright/mcp"]
}
```

**Capabilities:**

- Navigate web pages
- Take screenshots and snapshots
- Click, type, and interact with page elements
- Handle dialogs and file uploads
- Manage browser tabs

### memory

Knowledge graph for persistent memory across sessions.

```json
"memory": {
  "command": "cmd",
  "args": ["/c", "npx", "-y", "@modelcontextprotocol/server-memory"]
}
```

**Capabilities:**

- Create and manage entities
- Create relations between entities
- Add observations to entities
- Search and query the knowledge graph

### sequential-thinking

Step-by-step problem solving and analysis.

```json
"sequential-thinking": {
  "command": "cmd",
  "args": ["/c", "npx", "-y", "@modelcontextprotocol/server-sequential-thinking"]
}
```

**Capabilities:**

- Break down complex problems into steps
- Revise and branch thinking paths
- Generate and verify hypotheses
- Track multi-step reasoning

### filesystem

File system operations within the project directory.

```json
"filesystem": {
  "command": "cmd",
  "args": ["/c", "npx", "-y", "@modelcontextprotocol/server-filesystem", "."]
}
```

**Capabilities:**

- List directories
- Read files
- Create directory trees

### chrome-devtools

Chrome DevTools integration for debugging.

```json
"chrome-devtools": {
  "command": "cmd",
  "args": ["/c", "npx", "chrome-devtools-mcp@latest"]
}
```

**Capabilities:**

- Connect to Chrome DevTools Protocol
- Debug web applications
- Inspect network requests

### context7

Context management with configurable token limits.

```json
"context7": {
  "command": "cmd",
  "args": ["/c", "npx", "-y", "@upstash/context7-mcp@latest"],
  "env": {
    "DEFAULT_MINIMUM_TOKENS": "10000"
  }
}
```

**Configuration:**

- `DEFAULT_MINIMUM_TOKENS`: Minimum token count for context (set to 10000)

**Capabilities:**

- Fetch library documentation
- Resolve library IDs
- Context-aware documentation retrieval

## Usage

These MCP servers are automatically loaded by Claude Code when working in this project. No manual setup is required beyond having Node.js and npx available in your PATH.
